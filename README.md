# Моделирование фазовых переходов в ферромагнетиках методом Монте-Карло на основе модели Изинга

Модель Изинга --- это математическая модель, предназначенная для описания намагничивания материала.
Она представляет собой кристаллическую решётку, где в каждом узле находится спин. Он может быть направлен только 
вверх или вниз, что обозначается как +1 или -1.

Энергия системы описывается гамильтонианом:

$$
\mathcal{\hat{H}}=-J \sum\limits_{ \langle i, j \rangle}{s_i s_j}- B \sum\limits_i{s_i}
$$

В данном моделировании считаем, что внешнее магнитное поле B = 0

J - константа обменного взаимодействия соседних спинов i и j

Для моделирования фазового прехода создаем трехмерную систему n x n x n заполненный значениями 1 и -1 в рандомном порядке


Алгоритм Монте - Карло позволяет случайным образом "пробегать" по всем возможным конфигурациям спинов и находить состояние, 
соответствующее тепловому равновесию при заданной температуре (термолизация). Далее измеряем такие физические величины, как энергия, 
намагниченность, удельная теплоёмкость, удельная магнитная восприимчивость, и строим графики их зависимости от температуры
