энергия системы описывается гамильтонианом 
$$
H = -J sum_<i,j>{s_i s_j}  - B sum_i{s_i}
$$

принимаем, что внешнее магнитное поле B = 0

J - сила взаимодействия соседних спинов i и j

принимаем J = 1 (т.к. строим модель подходящуюю для всех веществ, а не для конкретного)

создаем трехмерный массив n x n x n заполненный значениями 1 и -1 в рандомном порядке

вычисляем энергию взаимодействия из гамильтониана
$$
E = -sum_<i,j>{s_i s_j}
$$
