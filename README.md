#Моделирование фазовых переходов в ферромагнетиках методом Монте-Карло на основе модели Изинга

энергия системы описывается гамильтонианом:

$$
H = -J \sum_{<i,j>}{s_i s_j}  - B \sum_i{s_i}
$$

считаем, что внешнее магнитное поле B = 0

J - сила взаимодействия соседних спинов i и j

считаем, что J = 1 (т.к. строим модель подходящуюю для всех веществ, а не для конкретного)

создаем трехмерный массив n x n x n заполненный значениями 1 и -1 в рандомном порядке

энергия взаимодействия спинов:

$$
E = -\sum_{<i,j>}{s_i s_j}
$$


реализуем алгоритм Монте - Карло с заданным количеством шагов для термолизации и вычисления характеристик


строим графики зависимости энергии, намагниченности, удельной теплоёмкости удельной магнитной восприимчивости от температуры
